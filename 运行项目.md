# 运行本项目

当前报错「无法将 npm 项识别为 cmdlet」说明 **本机尚未安装 Node.js**，或未加入 PATH。请先完成安装。

---

## 方法一：用 Windows 自带的 winget 安装（推荐）

在 **PowerShell** 或 **命令提示符** 里执行（需要管理员权限时选“以管理员身份运行”）：

```powershell
winget install OpenJS.NodeJS.LTS
```

安装完成后**关闭所有终端和 Cursor，再重新打开**，然后执行：

```powershell
cd c:\Users\hunte\Documents\Workspace\copy-of-lucky-baby-candy-party
npm install
npm run dev
```

---

## 方法二：从官网安装

1. 打开 **https://nodejs.org**
2. 下载并安装 **LTS** 版本
3. 安装时务必勾选 **“Add to PATH”**
4. 安装完成后**关闭并重新打开** Cursor 或终端，再执行：

```powershell
cd c:\Users\hunte\Documents\Workspace\copy-of-lucky-baby-candy-party
npm install
npm run dev
```

---

## 启动成功后

终端里会出现类似：`Local: http://localhost:3000/`  
在浏览器打开 **http://localhost:3000** 即可测试游戏。

---

## 若提示「在此系统上禁止运行脚本」

说明 PowerShell 限制了脚本执行。任选其一：

**A. 不改系统设置，直接用 cmd 版 npm（推荐）**

在 PowerShell 里执行（两行分开执行）：

```powershell
& "C:\Program Files\nodejs\npm.cmd" install
& "C:\Program Files\nodejs\npm.cmd" run dev
```

**B. 放宽当前用户的执行策略（一次设置长期有效）**

在 PowerShell 里执行：

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

执行后关闭并重新打开终端，再执行 `npm install` 和 `npm run dev` 即可。

---

## 若你确定已安装 Node，但终端仍找不到 npm

在 PowerShell 里执行下面命令，查找本机 `npm.cmd` 的位置：

```powershell
Get-Command npm -ErrorAction SilentlyContinue; dir "C:\Program Files\nodejs\npm.cmd" -ErrorAction SilentlyContinue; dir "$env:APPDATA\npm\npm.cmd" -ErrorAction SilentlyContinue
```

若某一行显示了路径，则用该路径运行，例如（把路径换成你查到的）：

```powershell
& "C:\Program Files\nodejs\npm.cmd" install
& "C:\Program Files\nodejs\npm.cmd" run dev
```
